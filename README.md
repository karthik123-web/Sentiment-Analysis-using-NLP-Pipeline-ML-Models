#  Sentiment Analysis using NLP & Machine Learning

This project was developed as part of my Data Science Internship (February 2026).

##  Overview
The goal of this project is to build a complete end-to-end sentiment analysis system. The model processes raw text data, applies NLP preprocessing, converts it into numerical features, and trains multiple machine learning models to classify sentiment.

## Steps Performed
- Data understanding and exploration
- Text preprocessing (cleaning, tokenization, stopword removal, lemmatization)
- Feature engineering using TF-IDF and Bag of Words
- Model training using:
  - Logistic Regression
  - Naive Bayes
  - Decision Tree
- Model evaluation using accuracy, precision, recall, and F1-score

##  Results
Logistic Regression achieved the best performance with highest accuracy and balanced metrics, making it the most suitable model for this task.

## Tools & Technologies
- Python
- Pandas, NumPy
- NLTK
- Scikit-learn

##  Conclusion
This project demonstrates how NLP preprocessing and feature engineering significantly impact model performance in text classification tasks.
